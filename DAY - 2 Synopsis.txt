Template Literals
Used for string interpolation

Syntax:

`Hello ${a}, ${b} and ${c}!`
Arrow Function
Short function syntax

Example:

let sum = (x, y) => x + y;
Array Functions

Map

Transforms each element
arr.map(x => x * 5);

Filter

Filters values based on condition
arr.filter(x => typeof(x) === "number");

Reduce

Reduces array to a single value
arr.reduce((acc, curr) => { /* operation */ }, acc);
Math Functions

Random Number (0–100)

Math.random() * 100;

Floor (1–10)

Math.floor(Math.random() * 10) + 1;

Ceil

Math.ceil(Math.random() * 100);
Type Casting
Convert string to integer
parseInt(val);
Array Methods

indexOf() → returns index

arr.indexOf("val");

includes() → returns boolean

arr.includes("val");

length → array size

arr.length;

push() → add to end

arr.push(val);

pop() → remove from end

arr.pop();

unshift() → add to beginning

arr.unshift(val);

shift() → remove from beginning

arr.shift();

Navigation:

window.location.href = "page_name";

id
class
selectors (querySelector, etc.)
