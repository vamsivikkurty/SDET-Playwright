1. Mathematical Operations

JavaScript supports basic arithmetic operations used for numerical calculations:

Addition (+): Adds two values → a + b
Subtraction (-): Subtracts one value from another → a - b
Multiplication (*): Multiplies two values → a * b
Division (/): Divides one value by another → a / b
Remainder (%): Returns the remainder of division → a % b
2. String Operations

Strings in JavaScript provide built-in methods for manipulation:

Original String: Stores text → str
Length: Returns number of characters → str.length
Uppercase Conversion: Converts to uppercase → str.toUpperCase()
Lowercase Conversion: Converts to lowercase → str.toLowerCase()
Substring Extraction: Extracts part of string → str.substring(0, 4)
3. Logical (Comparison) Operations

These operators compare values and return boolean results (true or false):

Greater than → a > b
Less than → a < b
Equal to → a == b
Not equal to → a != b
Greater than or equal to → a >= b
Less than or equal to → a <= b
4. Type Checking
The typeof operator is used to determine the data type of a variable
Example: typeof(str)
5. Objects

Objects store data in key-value pairs:

Example:

let obj = {
  name: "Alice",
  age: 30,
  role: "Trainer"
};
Used to represent structured data.
6. Functions

Functions are reusable blocks of code:

Syntax:

function nameOfTheFunc() {
    // code
}
They help in organizing and reusing logic.
7. Arrays

Arrays store multiple values in a single variable:

Example:

let arr = [1, 3, "Hey", true, 5];
let arr2 = [1, 2, 3, 4, 5];
Can hold mixed data types.
8. forEach Method

Used to iterate over array elements:

arr.forEach(function(x) {
    // code
});
Executes a function for each element.
9. Conditional Statements (If-Else)
Used to execute code based on conditions:
The if block checks a condition.
If false, control moves to the else block.
10. Switch Case

Used for multiple condition checks:

switch(choice) {
    case 1:
        // code
        break;
    default:
        // code
}
Provides a cleaner alternative to multiple if-else statements.
