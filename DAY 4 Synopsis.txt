
Promises

A Promise is used to handle asynchronous operations in JavaScript.
It represents a value that may be available now, later, or never.

function getData() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve("Data received");
        }, 2000);
    });
}

resolve() → called when the operation is successful
reject() → called when the operation fails

setTimeout() ->
Used to execute a function once after a specified delay.

setTimeout(() => {
    console.log("Hello");
}, 2000);
Runs only one time
Delay is in milliseconds (2000 ms = 2 seconds)

setInterval() ->
Used to execute a function repeatedly at fixed time intervals.

setInterval(() => {
    console.log("Data checking again and again");
}, 5000);
Runs continuously
Executes every 5 seconds (5000 ms)

API Fetch Synopsis (JavaScript Promises + Array Methods)

Uses fetch() to call different public APIs and retrieve data.
Converts responses using .json() to work with JavaScript objects.
Core Concepts Used
Chaining Promises → .then() for sequential processing
Error Handling → .catch() for failures

Array Methods:
map() → transform data

filter() → select specific data
